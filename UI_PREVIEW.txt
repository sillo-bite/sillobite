╔══════════════════════════════════════════════════════════════════════════════╗
║                          INTERACTIVE ELEMENTS                                 ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. Generate Code Button
   • Click → API call → Show loading → Display code
   • Hover: Darker purple background
   • Disabled during loading

2. Copy Button
   • Click → Copy to clipboard → Show checkmark → Toast notification
   • Hover: Light purple background
   • Icon changes: Clipboard → Checkmark (2 sec)

3. Generate New Code Button
   • Click → Invalidate old code → Generate new → Update display
   • Outline style button
   • Hover: Border highlight

4. Countdown Timer
   • Updates every second
   • Format: M:SS (e.g., 1:45)
   • Shows clock icon
   • Turns red when < 30 seconds
